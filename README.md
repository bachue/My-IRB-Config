# Installation

```bash
git clone git@github.com:bachue/My-Ruby-Config.git .ruby-config

mkdir -p $HOME/.bundle

ln -svf $HOME/.ruby-config/bundle_config $HOME/.bundle/config
ln -svf $HOME/.ruby-config/gemrc $HOME/.gemrc
ln -svf $HOME/.ruby-config/irbrc $HOME/.irbrc
ln -svf $HOME/.ruby-config/pryrc $HOME/.pryrc
ln -svf $HOME/.ruby-config/rspec $HOME/.rspec
```
